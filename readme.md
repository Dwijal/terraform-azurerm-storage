# azurerm/storage — Private Module

Creates an Azure Storage Account with optional blob containers,
network rules, soft delete, and blob versioning.

## Usage

```hcl
module "storage" {
  source  = "app.terraform.io/YOUR_ORG/storage/azurerm"
  version = "1.0.0"

  resource_group_name  = "rg-myapp-prod"
  location             = "eastus"
  storage_account_name = "stmyappprod001"
  account_tier         = "Standard"
  account_replication_type = "ZRS"

  enable_soft_delete         = true
  soft_delete_retention_days = 14

  containers = {
    "uploads"  = "private"
    "exports"  = "private"
  }

  tags = {
    environment = "prod"
    owner       = "platform-team"
  }
}
```
